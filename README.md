### ML-Based Healthcare Application

This project aims to predict the likelihood of diabetes in individuals using machine learning techniques, specifically classification algorithms. It was developed as part of an academic project to support early detection and promote health awareness.

---

##  Overview

Diabetes is a growing global health issue that requires early detection and proper management. Predicting diabetes risk using data-driven approaches can help individuals take preventive measures at the right time.

In this project, we:

- Collected and preprocessed healthcare dataset (PIMA Indian Diabetes Dataset)  
- Performed data cleaning and feature selection  
- Applied supervised machine learning for classification  
- Built and compared models for better accuracy  
- Evaluated performance using classification metrics  

---

##  Technologies Used

- Python 3.x  
- Twilio API  
- Ngrok Authentication  
- Pandas, NumPy – Data preprocessing  
- Scikit-learn – Model building and evaluation  
- Matplotlib, Seaborn – Data visualization  
- Flask – Web application integration  
- Jupyter Notebook – Model development  

---

##  Predicted Parameters

The model predicts whether a person is diabetic or not based on key health indicators such as:

- Glucose Level  
- Blood Pressure  
- Insulin  
- BMI (Body Mass Index)  
- Age  
- Skin Thickness  
- Diabetes Pedigree Function  

---

##  Model Performance

The model was evaluated using:

- Accuracy Score  
- Precision  
- Recall  
- Confusion Matrix  

The model achieved good accuracy and reliable prediction performance, making it useful for basic risk assessment.
